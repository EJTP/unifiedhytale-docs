## Projects

### List Projects

```
GET /v1/projects
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `type` | string | Filter by project type |
| `category` | string | Filter by category |
| `search` | string | Search in name/description |
| `sort` | string | Sort by: `newest`, `downloads`, `likes`, `updated` |
| `page` | number | Page number (default: 1) |
| `limit` | number | Results per page (default: 20, max: 100) |

**Response:**

```json
{
  "success": true,
  "data": [
    {
      "id": "abc123",
      "slug": "my-plugin",
      "name": "My Plugin",
      "summary": "A cool plugin",
      "type": "plugin",
      "downloads": 1234,
      "likes": 56,
      "createdAt": "2025-01-01T00:00:00Z",
      "updatedAt": "2025-01-15T00:00:00Z"
    }
  ],
  "meta": {
    "page": 1,
    "limit": 20,
    "total": 150
  }
}
```

### Get Project

```
GET /v1/projects/:slug
```

Returns full project details including description, versions, and dependencies.

### Get Project Versions

```
GET /v1/projects/:slug/versions
```

Lists all available versions for a project.

### Get Download URL

```
GET /v1/projects/:slug/download
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `version` | string | Specific version (default: latest) |

Returns a temporary signed download URL.

## Users


### Get User's Projects

```
GET /v1/users/:username/projects
```

## Licenses

### Verify License

```
POST /v1/licenses/verify
```

**Request Body:**

```json
{
  "key": "LICENSE-KEY-HERE",
  "projectId": "abc123"
}
```

**Response:**

```json
{
  "success": true,
  "data": {
    "valid": true,
    "userId": "user123",
    "projectId": "abc123",
    "expiresAt": "2026-01-01T00:00:00Z"
  }
}
```
