---
title: Editing Projects
description: How to edit and update your published projects.
---

After your project is published, you can still make changes to most details.

## What You Can Edit

### Always Editable
- **Name** - Change your project's display name
- **Summary** - Update the short description
- **Description** - Modify the full markdown description
- **Icon** - Upload a new project icon
- **Banner** - Update the header image
- **Gallery** - Add, remove, or reorder images
- **Categories** - Change categorization
- **Tags** - Add or remove tags
- **External Links** - Source code, wiki, etc.

### Cannot Be Changed
- **Project Type** - Plugin, World, etc.
- **Slug** - The URL identifier
- **Creation Date** - When first published
- **Owner** - Project ownership (contact support for transfers)

## Making Changes

1. Go to your [Dashboard](https://unifiedhytale.com/dashboard)
2. Click on the project you want to edit
3. Click **"Edit Project"**
4. Make your changes
5. Click **"Save Changes"**

<Callout type="info">
Most changes take effect immediately. Image updates may take a few minutes to propagate across CDN.
</Callout>

## Description Best Practices

When updating descriptions:

- Keep installation instructions current
- Note any breaking changes
- Link to changelogs for updates
- Include version compatibility info
- Credit contributors and dependencies

## Archiving Projects

If you want to stop maintaining a project:

1. Mark it as "Archived" in project settings
2. Users can still download existing versions
3. Project appears lower in search results
4. You can unarchive anytime

## Deleting Projects

<Callout type="warn">
Project deletion is permanent and cannot be undone.
</Callout>

To delete a project:
1. Go to project settings
2. Scroll to "Danger Zone"
3. Click "Delete Project"
4. Confirm with your password
