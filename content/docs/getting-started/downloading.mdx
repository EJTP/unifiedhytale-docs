All content on UnifiedHytale is virus-scanned and reviewed before being made available for download.

## How to Download

1. Navigate to any project page
2. Review the description and requirements
3. Check compatible Hytale versions
4. Click the **Download** button for your desired version
5. Follow installation instructions in the project description

## Understanding Dependencies

Some projects require other projects to work:

- **Required Dependencies** - Must be installed
- **Optional Dependencies** - Enhance functionality but not required
- **Incompatible** - Cannot be used together

Dependencies are listed on the project page and automatically checked during download.

## Installation

Installation varies by project type:

### Plugins
Place `.jar` files in your server's `plugins/` folder.

### Data Packs
Extract to your `datapacks/` folder.

### Asset Packs
Extract to your `assets/` folder.

### Worlds
Extract to your `worlds/` folder.

<Callout type="info">
Always read the project's installation instructions for specific requirements.
</Callout>

## File Security

Every file uploaded to UnifiedHytale is:

1. **Automatically Scanned** - Checked for viruses and malware
2. **Manually Reviewed** - Staff verifies content
3. **Hash Verified** - Ensures file integrity

<Callout type="warn">
Never download Hytale content from unofficial sources. Stick to trusted platforms like UnifiedHytale.
</Callout>

## Troubleshooting

**Download not starting?**
- Disable browser popup blockers
- Try a different browser
- Check your internet connection

**File corrupted?**
- Re-download the file
- Verify the file hash matches
- Contact support if issues persist
