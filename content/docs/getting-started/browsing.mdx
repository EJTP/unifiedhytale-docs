UnifiedHytale hosts thousands of mods, maps, resource packs, and more. Here's how to find exactly what you're looking for.

## Browse Page

Visit the [Browse page](https://unifiedhytale.com/browse) to explore all available content.

### Filtering Options

**By Type:**
- Plugins - Server-side Java code
- Data Packs - JSON configuration files
- Asset Packs - Models, textures, sounds
- Worlds - Complete playable saves
- Prefabs - Individual structures
- Modpacks - Curated collections

**By Category:**
- Gameplay - Mechanics, systems, features
- Aesthetic - Visual enhancements
- Technical - Performance, utilities
- Adventure - Quests, storylines
- Building - Construction tools

### Sorting

Sort results by:
- **Newest** - Most recently uploaded
- **Most Downloads** - Most popular overall
- **Most Likes** - Community favorites
- **Recently Updated** - Active projects

### Search

Use the search bar to find projects by:
- Project name
- Description keywords
- Author username

## Project Pages

Each project page shows:

- **Overview** - Description, screenshots, videos
- **Versions** - All available downloads
- **Dependencies** - Required other projects
- **Reviews** - Community feedback
- **Changelog** - Update history

## Following Projects

Click the **Follow** button to:
- Get notified of new versions
- Save to your collection
- Support the creator

## Server Listings

Find Hytale servers at [unifiedhytale.com/servers](https://unifiedhytale.com/servers):

- Browse by game mode
- See player counts
- Read server descriptions
- Connect directly from the listing
