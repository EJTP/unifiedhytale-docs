UnifiedHytale supports all Hytale modding formats. Here's an overview of each type.

## Plugin

Server-side Java code that extends Hytale functionality.

| Property | Value |
|----------|-------|
| Max Size | 200MB |
| Formats | `.jar`, `.zip` |
| Use Case | Custom game logic, commands, events |

**Examples:** Economy systems, minigames, anti-cheat

## Data Pack

JSON configuration files for game content.

| Property | Value |
|----------|-------|
| Max Size | 50MB |
| Formats | `.zip` |
| Use Case | Blocks, items, NPCs, world generation |

**Examples:** New weapons, custom mobs, biome configs

## Asset Pack

Visual and audio content for Hytale.

| Property | Value |
|----------|-------|
| Max Size | 100MB |
| Formats | `.zip` |
| Use Case | Models, textures, sounds, animations |

**Examples:** HD textures, custom models, sound replacements

## World

Complete playable world save files.

| Property | Value |
|----------|-------|
| Max Size | 500MB |
| Formats | `.zip` |
| Use Case | Adventure maps, minigame maps, showcases |

**Examples:** Survival maps, parkour courses, RPG worlds

## Prefab

Individual structures and buildings.

| Property | Value |
|----------|-------|
| Max Size | 50MB |
| Formats | `.zip` |
| Use Case | Houses, trees, dungeons, landmarks |

**Examples:** Medieval castle, modern house, cave system

## Modpack

Curated collection of compatible projects.

| Property | Value |
|----------|-------|
| Max Size | 500MB |
| Formats | `.zip` |
| Use Case | Pre-configured mod experiences |

**Examples:** RPG pack, survival enhanced, skyblock bundle

<Callout type="info">
Choose the most specific type for your project. If unsure, ask in our [Discord](https://unifiedhytale.com/discord).
</Callout>
